# حسابات مزايا المجتمع التجارية

نظام ويب بسيط لإدارة دخل ومصروفات المكتب بين الموظفين.

## الفكرة

- كل موظف يدخل برمز خاص به.
- الجميع يشاهد دخل ومصروفات المكتب بالكامل.
- الموظف يستطيع إضافة وتعديل وحذف عملياته فقط.
- مناسب للاستضافة المشتركة على Hostinger باستخدام PHP و MySQL.

## الموظفون الافتراضيون

- موسى: 4646
- يحيى: 3535
- كريم: 2727

> بعد رفع النظام وتشغيله، يمكن تغيير الرموز من قاعدة البيانات أو بإضافة صفحة إعدادات لاحقًا.

## التشغيل المحلي السريع

1. أنشئ قاعدة بيانات MySQL.
2. عدّل بيانات الاتصال في `config/database.php`.
3. افتح `install.php` مرة واحدة لإنشاء الجداول والموظفين.
4. احذف ملف `install.php` بعد نجاح التثبيت.
5. افتح `login.php` وسجّل الدخول برمز الموظف.
