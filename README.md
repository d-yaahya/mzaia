# حسابات مزايا المجتمع التجارية

نظام ويب بسيط لإدارة دخل ومصروفات المكتب بين الموظفين.

## الفكرة

- كل موظف يدخل برمز خاص به.
- الجميع يشاهد دخل ومصروفات المكتب بالكامل.
- الموظف يستطيع إضافة وتعديل وحذف عملياته فقط.
- مناسب للاستضافة المشتركة على Hostinger باستخدام PHP و SQLite بدون إعداد MySQL.

## الأمان

المستودع عام، لذلك لا يتم حفظ رموز الدخول داخل الكود. يتم إدخال رموز الموظفين أثناء التثبيت، ثم تخزينها مشفرة داخل قاعدة البيانات المحلية.

## التشغيل السريع على الجهاز

افتح PowerShell ثم نفّذ:

```powershell
cd D:\mzaia
php -S localhost:8000
```

ثم افتح المتصفح على:

```text
http://localhost:8000/install.php
```

بعد نجاح التثبيت احذف ملف `install.php` أو غيّر اسمه.

## التشغيل على Hostinger

ارفع الملفات إلى `public_html`، ثم افتح `install.php` مرة واحدة من المتصفح، وبعدها احذف ملف التثبيت.
