# حسابات مزايا المجتمع التجارية

نظام ويب بسيط لإدارة دخل ومصروفات المكتب بين الموظفين.

## المزايا

- دخول كل موظف برمز خاص.
- الجميع يشاهد كل العمليات والأرقام.
- الموظف يضيف ويعدل ويحذف عملياته فقط.
- ملخص دخل/مصروف/صافي اليوم.
- ملخص دخل/مصروف/صافي الشهر.
- ملخص شهري لكل موظف.
- فلترة العمليات حسب الفترة، الموظف، ونوع العملية.
- يعمل على Hostinger باستخدام PHP + SQLite بدون إعداد MySQL.

## التشغيل المحلي عبر PowerShell

```powershell
cd D:\mzaia
php -S localhost:8000
```

ثم افتح:

```text
http://localhost:8000/install.php
```

بعد التثبيت احذف ملف:

```powershell
Remove-Item .\install.php
```

## رموز الدخول

لا تحفظ الرموز داخل الكود. أثناء التثبيت أدخل:

- موسى
- يحيى
- كريم

ثم اكتب رموز الدخول الخاصة بهم في صفحة التثبيت. سيتم تخزينها مشفرة في قاعدة البيانات المحلية.
